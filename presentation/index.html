APPENDIX · Q&A Anticipated Questions

Defensible answers.

  • "Can it plug into real CAD?" — Via CAD plug-in APIs (CATIA / NX / SolidWorks all expose add-in APIs). Explicitly out of scope for today.
  • "What about false positives?" — The LLM can only fire with retrieved evidence from the company brain; every warning shows source △, report, and past outcome.
  • "Who are the competitors?" — CoLab AutoReview (closest, read-only) · Leo AI · CADDi · DraftAid. None have the write loop.
  • The Assignment — Collect 50 real drawings and reports, measure how often G-brain retrieval gives enough evidence for a useful LLM warning. That number decides where we stand vs. CADDi / CoLab.

Backup ladder: full live → recorded extraction → typewriter pseudo-animation. A working build always runs locally + tethering.
